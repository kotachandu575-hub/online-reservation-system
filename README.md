# Online Reservation System

## Overview

The Online Reservation System is a Java-based console application that allows users to reserve, view, and cancel reservations efficiently. The project demonstrates the use of Object-Oriented Programming (OOP) concepts, user authentication, and data management in Java.

## Features

* User Login Authentication
* Reservation Booking
* View Reservation Details
* Cancel Reservation
* Secure Logout
* Interactive Console Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* IntelliJ IDEA / Eclipse

## How to Run

1. Install Java JDK 21 or later.
2. Clone or download the project.
3. Open the project in IntelliJ IDEA or Eclipse.
4. Compile and run the main Java file.
5. Follow the on-screen instructions.

## Learning Outcomes

* Java Fundamentals
* OOP Concepts
* User Input Handling
* Exception Handling
* Console-Based Application Development
